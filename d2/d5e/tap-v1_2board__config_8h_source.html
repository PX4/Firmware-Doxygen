115 /*
116  * The slave address of the MPU-60X0 is b110100X which is 7 bits long.
117  * The LSB bit of the 7 bit address is determined by the logic level
118  * on pin AD0. This allows two MPU-60X0s to be connected to the same I2C bus.
119  * When used in this configuration, the address of the one of the devices
120  * should be b1101000 (pin AD0 is logic low) and the address of the other
121  * should be b1101001 (pin AD0 is logic high).
122  */
