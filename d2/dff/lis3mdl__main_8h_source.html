76 /**
77  * @brief Calibrate and self test. Self test feature cannot be used to calculate scale.
78  *
79  * SELF TEST OPERATION
80  * Note: To check the LIS3MDL for proper operation, a self test feature is incorporated :
81  * sensor offset straps are excited to create a nominal field strength
82  * (bias field) to be measured. To implement self test, the least significant bits
83  * (MS1 and MS0) of configuration register A are changed from 00 to 01 (positive bias).
84  * A few measurements are taken and stored with and without the additional magnetic
85  * field. According to ST datasheet, those values must stay between thresholds in order
86  * to pass the self test.
87  */
