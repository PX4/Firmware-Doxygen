Detailed Description

Driver for the PCA9685 I2C PWM module The chip is used on the Adafruit I2C/PWM converter https://www.adafruit.com/product/815.

Parts of the code are adapted from the arduino library for the board https://github.com/adafruit/Adafruit-PWM-Servo-Driver-Library for the license of these parts see the arduino_Adafruit_PWM_Servo_Driver_Library_license.txt file see https://github.com/adafruit/Adafruit-PWM-Servo-Driver-Library for contributors
