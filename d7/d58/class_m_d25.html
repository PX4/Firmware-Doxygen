enum MD25::e_mode

modes

NOTE: this driver assumes we are always in mode 0!

seprate speed mode: motor speed1 = speed1 motor speed2 = speed2 turn speed mode: motor speed1 = speed1 + speed2 motor speed2 = speed2 - speed2 unsigned: full rev (0), stop(128), full fwd (255) signed: full rev (-127), stop(0), full fwd (128)

modes numbers: 0 : unsigned separate (default) 1 : signed separate 2 : unsigned turn 3 : signed turn
