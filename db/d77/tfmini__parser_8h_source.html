45 // Data Format for Benewake TFmini
46 // ===============================
47 // 9 bytes total per message:
48 // 1) 0x59
49 // 2) 0x59
50 // 3) Dist_L (low 8bit)
51 // 4) Dist_H (high 8bit)
52 // 5) Strength_L (low 8bit)
53 // 6) Strength_H (high 8bit)
54 // 7) Reserved bytes
55 // 8) Original signal quality degree
56 // 9) Checksum parity bit (low 8bit), Checksum = Byte1 + Byte2 +...+Byte8. This is only a low 8bit though
