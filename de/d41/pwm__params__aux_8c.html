PARAM_DEFINE_INT32 ( PWM_AUX_RATE  ,
50   
)

Set the PWM output frequency for the auxiliary outputs.

Set to 400 for industry default or 1000 for high frequency ESCs.

Set to 0 for Oneshot125.

true

-1 2000 Hz PWM Outputs

PARAM_DEFINE_INT32 ( PWM_AUX_MIN  ,
1000   
)

Set the minimum PWM for the auxiliary outputs.

Set to 1000 for industry default or 900 to increase servo travel.

true

800 1400 us PWM Outputs

PARAM_DEFINE_INT32 ( PWM_AUX_MAX  ,
2000   
)

Set the maximum PWM for the auxiliary outputs.

Set to 2000 for industry default or 2100 to increase servo travel.

true

1600 2200 us PWM Outputs

PARAM_DEFINE_INT32 ( PWM_AUX_DISARMED  ,
1500   
)

Set the disarmed PWM for auxiliary outputs.

This is the PWM pulse the autopilot is outputting if not armed. The main use of this parameter is to silence ESCs when they are disarmed.

true

0 2200 us PWM Outputs
